# Projeto: Painel de Temperatura de Piscina

Este projeto foca na criação de um sistema de monitoramento de temperatura em rede local.

- [x] Painel de exibição dupla simultânea.

- [x] Autenticação individual (Login/Senha).
- [x] Armazenamento automático de metadados (Técnico, Data, Hora).
- [x] Relatórios semanais e mensais por técnico.
- [x] Atualização automática em tempo real.
